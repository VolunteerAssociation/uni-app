# uni-app 商城购物App
- 商城购物网站

## 软件架构
- 软件架构说明，通过uni-app框架vue3塔建的项目结构

## 项目依赖
- HBuilderX(v3.7.11以上)
  - 插件依赖
  - uni-app（Vue3）编译
  - less
  - sass
  - uni_modules
- uni-ui(v1.4.27以上)

## 目录结构
```
├─.hbuilderx
├─api                                   // 请求接口
  ├─home
  ├─login
  ├─mine
  ├─order
  ├─BASE_URL.js
  └─request.js
├─components                            // 组件
  ├─HeaderDropdown
  └─RightContent
├─page                                  // 目录
  ├─home
  ├─login
  ├─mine
  ├─order
  └─register
├─static                                // 静态资源
├─stores                                // 状态管理
  └─counter.js
├─uni-modules
├─utils                                 // 工具库
  └─utils.js
├─.gitignore
├─App.vue
├─index.html
├─LICENSE
├─main.js
├─manifest.json
├─pages.json
├─README.md
└─uni.scss
```
